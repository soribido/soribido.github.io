# 리눅스 명령어 정리(update 2022.11.11)
```bash
bash
```
shell (리눅스 기본 쉘 bash) 실행. 터미널 기본으로 bash 설정 안되어 있을 때 실행.

```bash
sudo
```
root 권한으로 명령 실행. 처음에 한해 사용자 계정 비밀번호 입력해야 할 수 있음.
/etc/sudoers 파일을 수정하여 사용자 계정이나 그룹에 권한 부여 가능.
ex) sudo su, sudo apt-get update

```bash
ls
```
파일 목록 출력

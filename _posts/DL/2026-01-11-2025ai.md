### Anthropic
* `2025.02` Claude Code 베타 출시   
    CLI를 통해 다양한 환경에서 작동하게 출시했다. 장단점이 있지만 돌아보면 성공적인 판단이었던 것 같다.  
* `2025.05` Claude Code 5월 정식 출시  
* `2025.08` Claude for Chrome 베타 출시  
* `2025.09` Claude 파일(ppt, pdf, excel등) 생성 기능 출시  
* `2025.10` Claude Skills 출시.   
    md파일(명세)이 있으면 domain specific한 지식을 전달해줄 수 있다. 예를 들면 ppt작성 전문가가 될 수 있다. 스킬은 lazy loading으로 처음 시작할 때 컨텍스트를 다 먹는게 아니고 일부만 가져왔다가 필요할 때 더 읽어서 쓰는 식이다. 
* `2025.11` Claude in Excel 출시  
    엑셀 프로그램 내부에 claude가 들어가면서 자동화 작업이 쉬워졌다고 한다.
* `2025.12` Claude Code 매출 1조원 달성 / Bun 인수  
    Claude code는 내부적으로는 bun실행파일로 작동하는데 안정성을 위해 오픈소스 프로젝트인 bun 인수했다.
* `2025.12` Slack에서 사용하는 Claude / Chrome 확장 프로그램 정식 출시  
    모든 유료 플랜 구독자는 Claude for Chorme 확장 프로그램을 사용할 수 있다.
* `2025.12` MCP를 AAIF(Agentic AI Foundation, 리눅스 재단 산하)에 기부하여 업계 오픈 표준화

### OpenAI

    브라우저를 직접 제어할 수 있는 기능이다.

    MCP는 이전에 Anthropic이 도입한 개념이다. OpenAI에서도 이 개념을 채택하였다.
* `2025.05` Codex preview (베타 출시)  
   내부적으로는 openai가 가상 컨테이너를 웹에다가 생성해서 소통하는 형식이라 실 사용 측면에서 latency가 좀 있다.
* `2025.07` Operator -> Chatgpt Agent로 편입
* `2025.08` Codex CLI 출시
* `2025.08` GPT5 출시
* `2025.09` Sora2 출시, 비디오 생성 모델과 ios앱 출시
* `2025.10` chatgpt 전용 브라우저 Atlas 출시  
    mac만 가능하다고 한다.
* `2025.11` GPT5.1 출시  
    GPT5의 개선 버전이다. 여담으로 1주일뒤에 gemini 3.0이 출시된다.  
    이후 내부적으로 '코드 레드'(비상체제)가 선포된다.
* `2025.12` GPT5.2 출시  
    시간 순서상 코드 레드 이후 출시된 모델이다. gemini, claude 등 모든 모델이 마찬가지이지만 아직 고난도의 수학문제나 이미지를 해석하는 능력은 떨어진다.
* `2025.12` Agents.md AAIF에 기부  
    agents.md 는 모델을 위한 README느낌이다.  
    cluade.md가 프로젝트에 관한 것이고 readme.md가 사람을 위한 것이라면 agents.md는 모델을 위한 것이다.   
    ex.어떤 파일 수정해도 되는지 등등.. https://agents.md/ 를 참고.
* `2025.12` Codex Skills 출시  
    Anthropic에서 출시한 skills 개념을 받아들였다.
* `2025.12` Disney x Sora 콘텐츠 파트너십 체결

### Google
* `2025.02` Gemini2.0
* `2025.03` Gemini2.5 Thinking 모델 첫 공개
* `2025.03` Google jules 베타 출시
    github 레포지토리랑 연동하여 작업이 가능하며 google cloud vm에 코드를 클론하여 작업을 수행할 수 있는 프로젝트이다. 특징은 3~4시간이 걸리는 코드 작업도 계속 수행할 수 있다는 점. 개인적인 생각으로는 claude code랑 다른 방향으로 접근하려고 했던 것 같고 이후에 codex가 jules에 영향을 받지 않았나 추측한다. 테스트 당시 코드 자체에는 오류를 발생시키는 코드가 많이 나오긴 했으나 고난도의 작업이기도 했고 앞으로의 개선을 기대했는데 지금은 대부분 claude code로 작업하는 듯 하다.
* `2025.04` NotebookLM Audio  
    `참고` NotebookLM은 RAG처럼 문서를 넣으면 그 문서를 기반으로 자료를 만든다던지 할 수 있는 구글의 플랫폼이다.
* `2025.05` Google I/O 2025  
    Veo3, Imagen 4 발표
* `2025.05` Google AI studio 웹앱 빌더 출시  
    '게임 만들어줘'같은 요청을 실제로 웹앱으로 만들 수 있다.  
    다만 연말에 토큰 관련 보안이슈 터진다. (AI studio로 배포하면 토큰을 알 수 있는 큰 보안 이슈)
* `2025.06` Gemini CLI  
    gemini도 cli의 흐름에 합류한다.
* `2025.07` NotebookLM Video Overviews 기능 추가 
    강의 영상 생성 등에 잘 활용되고 있다고 한다.
* `2025.08` Google jules 정식 출시
* `2025.08` 월드 모델 Genie3 데모 (LMArena)  
    아직 갈 길이 먼 분야긴 하지만 대단하다는 생각이 든다. 출시 당시에는 모델의 기억력이 길지 않은 단점이 있었다. (앞을 보고 뒤로 보면 있었던 물체가 없어지는 경우)
* `2025.08` NanoBanana (LMArena)  
    나노바나나라는 의문의 이미지 생성 모델이 등장하며 큰 화제를 모은다. 정식 명칙은 Gemini 2.5 Flash Image이다. 기존 타 모델 대비 일관성 유지에 독보적이며 이때의 파급력이 워낙 커서 지금도 유저들에게는 나노바나나라 불린다. 개인적으로 이전까지의 이미지들은 AI로 만든 것인지 어느정도 구별이 된다는 입장이었는데 깜짝 놀랄 정도의 퀄리티였으며 지인들도 전혀 구별 못하는 모습을 보여줬다.
